# astrbot_plugin_64gua
## 周易六十四卦随机生成器
### 发送指令（/金钱卦）就可以触发，需要下载金钱卦64卦本地图片资源。
### 金钱卦资源链接：https://www.123865.com/s/wumovd-F5dAh
### 把资源包解压后放在插件同一目录即可，即/plugins/astrbot_plugin_64gua目录下。
### 只测试过QQofficial和napcat平台，其他平台可能会有错误。本插件采用ai编写，如有bug，本人计算机小白，一概不知。
